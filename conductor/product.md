# Product Definition: Pembimbing Andalus

## Overview

A simple, responsive, and easy-to-use web application for displaying a list of room supervisors ("Pembimbing Kamar").

## Core Vision

The vision of Pembimbing Andalus is to provide a clean, modern, and highly accessible interface for viewing and managing room supervisor data. It aims to bridge the gap between simple data tables and interactive dashboards by offering multiple views (Table and Card) and helpful utilities like a dashboard summary and a copy-to-clipboard feature.

## Target Audience

- **School Administrators**: To quickly view and verify the list of supervisors.
- **Students & Parents**: To easily find information about room supervisors.
- **Staff**: For daily operational use within the "Andalus" environment.

## Key Features

- **Dual Display Modes**: Toggle between a structured **Responsive Table** for desktop/tablets and a modern **Card View** (Grid) optimized for mobile devices.
- **Grouping & Accordion Logic**: Supervisors are organized by room category, with expandable/collapsible sections to keep the interface clean.
- **Live Search & Filter**: Instant filtering by name or role to find specific supervisors without page reloads.
- **Dashboard Summary**: A visual header showing key statistics such as total supervisors and total rooms managed.
- **One-Click Utilities**: Integrated copy-to-clipboard functionality for supervisor names and roles to streamline communication.

## Success Metrics

- **Performance**: Near-instant load times with no external dependencies (Vanilla JS/CSS).
- **Accessibility**: Fully responsive design that works seamlessly across all devices.
- **Simplicity**: No build steps required; a "pure web" experience that is easy to host (like on GitHub Pages).
