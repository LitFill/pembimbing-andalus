# Technology Stack: Pembimbing Andalus

## Core Languages

- **HTML5**: Semantic markup for the core structure.
- **CSS3**: Modern CSS for styling, including custom properties (CSS variables) for flexible theme management.
- **JavaScript (Vanilla)**: All core logic is implemented in modern, standard-compliant JavaScript without the use of external frameworks or libraries.

## Frontend Tools

- **Static Assets**: No preprocessors or build tools (like Sass or PostCSS) are currently in use, ensuring the project remains portable and lightweight.

## Deployment & Hosting

- **GitHub Pages**: The project is configured for continuous deployment using GitHub Actions, serving static content directly from the `main` branch.

## Environment Requirements

- **Browser Compatibility**: Target modern web browsers (Chrome, Firefox, Safari, Edge) that support ES6+ features and standard CSS layouts.
